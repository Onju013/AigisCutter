============================================
アイギスCutter ver2.0.0
============================================

◆はじめに
　　DMMで配信されているゲーム「千年戦争アイギス」において、
　　スクリーンショットからゲーム領域を切り取るツールです。

　　wiki編集のため自分用として作成したのを公開したものです。
　　（さすがに初期のやっつけツールよりは真面目なものになりつつありますが）

◆実行方法
　　当アプリケーションを実行するためには .NET Framework 4.6.1 が
　　インストールされている必要があります。

　　AigisCutter.exeを実行してください。
　　アンインストールしたい場合はファイル一式を削除するだけで大丈夫です。　　

◆使い方
　　１．ウィンドウ上部のリストボックスに画像ファイル（複数可）をドラッグ＆ドロップ
　　２．「抽出開始！」ボタンを押下
　　３．○○○_cut.pngというファイルが出来るので中身を確認
　　　　領域が判定出来ない場合はファイルが作成されません

　　「Clear」ボタン：リストボックスをクリア（ファイルは消えません）
　　「ゴミ箱へ」ボタン：元ファイルをゴミ箱へ移動
